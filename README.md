## Run Locally

Clone the project

```bash
  git clone https://github.com/RusssianMusketeer/Personal-Portfolio.git
```

Go to the project directory

```bash
  cd Personal-Portfolio/
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
